==================================================================================================================
                                              RunnableLambda
==================================================================================================================

RunnableLambda is used when you want to wrap a Python function and make it usable inside a LangChain pipeline.

To add custom code in a LangChain

==================================================================================================================
                                              RunnablePassthrough
==================================================================================================================

RunnablePassthrough is a special runnable that does nothing to the input.
Acts like a bridge that forwards input to the LLM.


==================================================================================================================
                                              Runnablebranch
==================================================================================================================

RunnableBranch allows you to run different runnables based on a condition.
Think of it as an if-else or switch-case for pipelines.
